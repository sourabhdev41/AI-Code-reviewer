# Code Reviewer

A basic frontend code reviewer built using HTML, CSS, and JavaScript.

## Features
- Paste Python code
- Basic issue detection
- Clean UI

## Tech Stack
- HTML
- CSS
- JavaScript

## Future Improvements
- Connect with Python backend
- Add AI-based review
- Syntax highlighting

## How to Run
1. Download or clone repo
2. Open index.html in browser
